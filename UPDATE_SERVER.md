## Настройка webhook в Tilda:

1. Зайти в настройки сайта в Tilda
2. Перейти в раздел "Формы" → "Webhook"
3. Указать URL: `https://app.e-budget.ru/api/webhook/tilda`
4. Сохранить настройки
5. В настройках формы на странице включить чекбокс "WEBHOOK"
6. Опубликовать страницу

## Проверка работы:

После обновления webhook должен успешно подключиться в Tilda без ошибки "Invalid email".

При отправке реальной формы с данными:
- Создастся/обновится пользователь в таблице Users
- Если это покупка - создастся запись в Purchases
- Начислятся бонусы
