## 💻 Вариант 2: Запустить локально (15 минут)

### Зачем?
- Протестировать систему
- Понять как работает
- Настроить под себя

### Как?

#### Шаг 1: Запустить Docker
```bash
docker-compose up -d
```

#### Шаг 2: Настроить NocoDB
Открыть: **http://localhost:8080**
- Создать аккаунт
- Импортировать CSV файлы
- Получить API токен

#### Шаг 3: Настроить .env
```bash
notepad backend\.env
```
Вставить токен и Project ID

#### Шаг 4: Перезапустить
```bash
docker-compose restart api
```

#### Шаг 5: Проверить
Открыть: **http://localhost:3000/health**

**Подробнее:** [QUICKSTART.md](QUICKSTART.md)

## 🌐 Вариант 3: Задеплоить на продакшн (1-2 часа)

### Зачем?
- Использовать в реальном проекте
- Подключить к Tilda
- Запустить для клиентов

### Как?

#### Шаг 1: Арендовать VPS
- DigitalOcean, Hetzner, или любой другой
- Минимум: 2GB RAM, 1 CPU

#### Шаг 2: Установить Docker
```bash
curl -fsSL https://get.docker.com -o get-docker.sh
sh get-docker.sh
```

#### Шаг 3: Загрузить проект
```bash
git clone https://github.com/ВАШ_USERNAME/e-budget-backend.git
cd e-budget-backend
```

#### Шаг 4: Настроить и запустить
```bash
cp backend/.env.example backend/.env
nano backend/.env  # Настроить
docker-compose up -d
```
