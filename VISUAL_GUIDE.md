## 🔄 Поток данных

### Сценарий 1: Пользователь открывает личный кабинет

```
1. User → Tilda Page
   └─> Tilda Members проверяет авторизацию
   
2. Tilda → localStorage
   └─> Сохраняет profile с email
   
3. JS Widget → localStorage
   └─> Читает user_email
   
4. JS Widget → API Server
   └─> POST /api/widget/courses
       { "user_email": "user@email.com" }
   
5. API Server → NocoDB
   └─> GET /api/v1/db/data/v1/Users?where=(Email,eq,user@email.com)
   
6. NocoDB → PostgreSQL
   └─> SELECT * FROM users WHERE email = 'user@email.com'
   
7. PostgreSQL → NocoDB → API Server → JS Widget → User
   └─> Отображает таблицу с курсами
```

### Сценарий 2: Пользователь покупает курс

```
1. User → Tilda Payment Form
   └─> Заполняет форму и оплачивает
   
2. Tilda → Webhook
   └─> POST https://your-domain.ru/api/webhook/tilda
       {
         "Email": "user@email.com",
         "order_id": "1234567890",
         "payment": "Да",
         "course_id": "recXXXXXX",
         "amount": 420
       }
   
3. API Server → NocoDB
   └─> Создаёт покупку
   └─> Начисляет бонусы покупателю (300₽)
   └─> Начисляет бонусы рефералу (100₽)
   
4. NocoDB → PostgreSQL
   └─> INSERT INTO purchases ...
   └─> UPDATE users SET bonuses = bonuses + 300 ...
   
5. API Server → Tilda
   └─> { "success": true, "user_id": "recXXXXXX" }
```

### Сценарий 3: Пользователь приглашает друга

```
1. User → Личный кабинет
   └─> Видит реферальную ссылку
   
2. JS Widget → API Server
   └─> POST /api/widget/referrals
       { "user_email": "user@email.com" }
   
3. API Server → NocoDB
   └─> Получает пользователя
   └─> Генерирует реферальную ссылку
   └─> Получает список рефералов
   
4. API Server → JS Widget
   └─> Возвращает HTML с:
       • Реферальной ссылкой
       • Количеством рефералов
       • Заработанными бонусами
   
5. User → Копирует ссылку
   └─> Отправляет другу
   
6. Friend → Переходит по ссылке
   └─> utm_source=recXXXXXX (ID реферала)
   
7. Friend → Регистрируется
   └─> Webhook → API Server
   └─> Создаёт пользователя с полем "Кто привёл"
   └─> Обновляет счётчик рефералов
```

## 💰 Бонусная система

```
Событие: Покупка курса
│
├─> Покупатель получает 300₽
│   └─> UPDATE users SET bonuses = bonuses + 300
│
└─> Если есть реферал
    └─> Реферал получает 100₽
        └─> UPDATE users SET bonuses = bonuses + 100

Событие: Регистрация по реферальной ссылке
│
├─> Новый пользователь создаётся
│   └─> INSERT INTO users (email, referrer_id, ...)
│
└─> Счётчик рефералов обновляется
    └─> UPDATE users SET referral_count = referral_count + 1
```
